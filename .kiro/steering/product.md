# Product Overview

PDF Tools is a privacy-focused, client-side web application for manipulating PDF files. The tool operates entirely in the browser with zero server communication, ensuring complete data privacy.

## Core Features

- **Merge Mode**: Combine multiple PDF files (up to 200MB each) into a single document
- **Split Mode**: Divide PDFs (up to 1GB) using three methods:
  - Individual pages (one file per page)
  - Custom page ranges (e.g., "1-3, 5-7, 9")
  - Every N pages (split at regular intervals)
- **Compress Mode**: Reduce PDF file size (up to 1GB) by converting pages to JPEG images with adjustable quality presets (low/medium/high) or custom quality settings

## Key Value Propositions

- 100% offline processing - files never leave the user's device
- No file uploads to servers
- No data collection or tracking
- Works entirely in the browser
- No installation required
- Free and open source

## Target Users

Users who need to manipulate PDFs while maintaining complete privacy and security, including professionals handling sensitive documents, students, and anyone concerned about data privacy.
